## Getting Started

Install all dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

# Tastings App

## Migrations

Run the migrations:

```
npx prisma migrate dev --name migration_name
```

Apply to the prisma client:

```
npx prisma generate
```
